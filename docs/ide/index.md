IDEA
===
导航 Navigation
===
|按 Press|功能 Function|
|  ----  | ----  |
|Ctrl+Alt+S|网易有道, 设置热键|
|Shift+F6|微软拼音输入法, 常规/使用原先输入法|
|**Ctrl+N**|可以快速打开类, 查找类|
|**Ctrl+G**|行号定位代码|
|**Ctrl+Alt+左右箭头**|快速回到上次打开的代码文件, 返回至上次浏览的位置|
|**Ctrl+E**|最近的文件|
|**Ctrl+Alt+L**|格式化代码|
|**Ctrl+Alt+I**|将选中的代码进行自动缩进编排，这个功能在编辑 JSP 文件时也可以工作|
